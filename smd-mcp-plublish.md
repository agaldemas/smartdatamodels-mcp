### 🚀 New Tool: Connect AI Agents to Smart Data Models with the MCP Server

We are proud to announce the release of the __Smart Data Models MCP Server__! This new open-source tool brings the power of the Model Context Protocol (MCP) to the FIWARE ecosystem, allowing AI agents like __Claude__ and __Cline__ to seamlessly interact with all smart data models.

__What can you do with it?__

- __🔍 Instant Discovery:__ Ask your AI to find models across 15+ domains (Smart Cities, Energy, Agrifood...) without leaving your chat.
- __⚡ JSON to NGSI-LD:__ Automatically convert raw JSON data into NGSI-LD compliant entities.
- __✅ Smart Validation:__ Validate your data payloads against official schemas instantly.
- find some matching model from json data
It’s a game-changer for developers building IoT platforms and Digital Twins.
